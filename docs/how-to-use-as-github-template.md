# How to use this repository as a GitHub template

1. Click **Use this template** in GitHub.
2. Create a new repository from this template.
3. Clone the new repository locally.
4. Run `pnpm install`.
5. Generate an assignment scaffold with:
   - `pnpm create-assignment --name my-react-assignment --framework react --preset funnel`
   - or interactive mode: `pnpm create-assignment`
6. Verify scaffold integrity with `pnpm verify:scaffold`.
