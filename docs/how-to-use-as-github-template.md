# How to use this repository as a GitHub template

1. Click **Use this template** in GitHub.
2. Create your own repository from this template.
3. Clone and install dependencies.

```bash
pnpm install
```

4. Generate your first assignment scaffold.

```bash
pnpm create-assignment --name checkout-funnel --framework react --preset funnel
```

5. Run scaffold verification before pushing.

```bash
pnpm verify:scaffold
```
