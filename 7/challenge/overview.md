# Challenge Overview

7회차 과제는 drag-and-drop 칸반 보드입니다.

핵심은 "카드를 옮긴다"가 아니라 다음 질문에 답하는 것입니다.

- 사용자가 본 drag 결과를 어떤 서버 명령으로 바꿀 것인가?
- 저장 응답을 기다리는 동안 UI는 무엇을 보여줄 것인가?
- 실패하거나 충돌하면 어떤 snapshot으로 되돌릴 것인가?
- 선택 카드와 필터는 URL로 어떻게 복원할 것인가?
- mouse가 아닌 사용자도 같은 작업을 할 수 있는가?

제공된 코드는 이 질문을 풀기 위한 scaffold입니다. 정답 구현은 들어 있지 않습니다. 화면은 보드 형태로 뜨지만, drag-and-drop과 저장/복구 로직은 멘티가 작성해야 합니다.
